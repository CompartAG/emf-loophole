EMF Loophole Releases Repository

EMF Loophole is released in several versions, one for each EMF version. This is an Eclipse Update site/p2 repository providing all the releases of these versions. However, you must access it from Eclipse by using the following URL:

http://mbarbero.github.io/emf-loophole

Individual versions for each EMF version can be a accessed with the following URLs:

  • http://mbarbero.github.io/emf-loophole/2.5 for EMF 2.5
  • http://mbarbero.github.io/emf-loophole/2.6 for EMF 2.6
  • http://mbarbero.github.io/emf-loophole/2.7 for EMF 2.7
  • http://mbarbero.github.io/emf-loophole/2.8 for EMF 2.8
  • http://mbarbero.github.io/emf-loophole/2.9 for EMF 2.9
